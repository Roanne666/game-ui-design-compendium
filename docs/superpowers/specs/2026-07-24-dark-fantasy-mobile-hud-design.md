# 02 Dark Fantasy — 竖屏触控魂类 HUD 设计

日期: 2026-07-24  
状态: 已确认（对话确认）

## 目标

在 `demo/scenes/02-dark-fantasy/` 交付与 Style 01 同完成度的**可玩竖屏 HUD**，体现 Dark Fantasy / 魂类哲学：极简、古金、触控拇指区、默认 9:20。

## 画幅与输入

- 画幅: 9:20 居中手机框
- 主输入: 触控（拖动环视、点目标、底栏大键）
- 禁止: 键鼠主帮助文案；共享 `components.css` 网页壳作为最终 HUD

## 竖屏信息架构

| 区域 | 内容 |
|------|------|
| 底中偏左 | HP / FP / Stamina 短条；数值变化时亮起，约 2s 后淡出 |
| 顶中 | 地点名（弱）；靠近篝火时「Rest」提示 |
| 右下 | ATTACK 大键 + ROLL / USE |
| 左下 | 按住 LOCK |
| 底中 | Estus 等道具格 |
| 中央偏下 | 篝火石板短菜单（Rest / Leave） |

## 视觉

- 色: `#C9A961` 古金、`#E8DCC4` 骨白、`#0A0908` 近黑、低血 `#8B0000`
- 字: Cinzel / Cormorant Garamond
- 无霓虹、无扫描线、无大面积发光

## 玩法循环

1. 拖动环视；点敌人锁定（或按住 LOCK）
2. ATTACK 耗耐力；ROLL 翻滚耗耐力
3. 受击掉血；低血条变血红
4. 点 Estus 回血（有限次数）
5. 靠近篝火 → 石板菜单 Rest 回满并补充 Estus

## 非目标

- 不做赛博方言切换
- 不做完整装备/属性页
- 不做横屏四角 PC HUD 移植
