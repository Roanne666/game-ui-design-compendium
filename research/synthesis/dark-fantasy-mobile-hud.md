## 映射表（通用层 → 02）

| 通用分区 | 02 落地 |
|----------|---------|
| 顶信息 | 宁姆格福 · 废墟；近篝火休息提示 |
| 帮助 | 顶右图标；指引展开在顶区 |
| 情境提示 | 「点按「互动」· 篝火」等 |
| 中下左短状态 | 生命 / 集中 / 耐力（变化浮现） |
| 中下中道具 | 露滴瓶图标 + 数量角标 |
| 左下辅键 | 锁定（按住） |
| 右下主键 + 竖列次要 | 攻击；上叠翻滚、互动 |
| 中央模态 | 篝火石板：休息 / 离开（保留中文） |

## 材质与图标

- 克制魂系：双层古金边、角饰、金属内凹底（见 `hud.css`）
- 触控图标：`art/icons/` 项目生成暗金浮雕（lock / use / roll / attack / estus / help）
- 键面无汉字；中文在 `aria-label`、提示、石板、Feed

## 02 踩过的坑（已写进通用反例）

1. 帮助文与露滴同用底栏高度 → 遮挡  
2. 次要键伸进中央 → 与道具打架  
3. 生命条过宽 → 叠露滴  
4. 键上写「攻击」等 → 网页感  
5. 终稿应用生成材质图标，而非仅线框 SVG  

## 验收循环（02）

拖动环视 → 锁定/攻击/翻滚 → 露滴回血 → 近篝火互动 → 石板休息。
