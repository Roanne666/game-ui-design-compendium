# /pick-style

根据用户描述的游戏类型，从本库 10 套竖屏 UI 风格中推荐 **一个** pack。

## 用法

用户提供：游戏大类、核心玩法、或参考作品（如「魂类」「JRPG 菜单」「太空舰桥」）。

## 步骤

1. 读 `references/style-matrix.md`
2. 对照「类型 / 布局原型 / 核心玩法循环」匹配最接近的一行
3. 输出推荐结果（仅一个 pack）

## 输出格式

```markdown
## 推荐风格

- **Pack ID**：`NN-style-name`
- **路径**：`library/packs/NN-style-name/`
- **理由**：（一句话：为何该行的布局与玩法匹配用户需求）
- **核心循环**：（从矩阵摘抄该行玩法，便于用户确认）

若需生成 UI，请使用 `/apply-style` 并指定上述 pack id。
```

## 约束

- 只推荐一个 pack；不要列出全部 10 套
- 若用户类型明确落在矩阵多行，选最贴近核心玩法的一行并说明取舍
- 不推荐 `research/raw` 作为依据
